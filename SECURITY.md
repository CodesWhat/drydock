# Security Policy

## Supported Versions

| Version | Supported |
| --------- | -------------------- |
| latest | :white_check_mark: |

## Reporting a Vulnerability

**Do not open a public GitHub issue for security vulnerabilities.**

Please use [GitHub's private vulnerability reporting](https://github.com/CodesWhat/drydock/security/advisories/new).

We appreciate responsible disclosure and will credit reporters in the release notes (unless you prefer to remain anonymous).
